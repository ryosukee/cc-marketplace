# バージョン更新通知スキップ

SessionStart hook のバージョン更新通知を「確認済み」にして、以降の通知を抑制する。

現在のバージョンを `last-version` に記録することで、次回 SessionStart 時に差分がなくなり通知が消える。

重要: このスキルはメインセッションでのみ実行する。Agent ツールで起動されたサブエージェント、チームエージェント、または `--agent` フラグで起動されたカスタムエージェントでは、このスキル全体をスキップすること。

## ワークフロー

### ステップ 1: 現在のバージョンを取得

Bash で以下を実行:
```
claude --version 2>/dev/null | awk '{print $1}'
```

### ステップ 2: バージョンを記録

ステップ 1 で取得したバージョンを記録する:
```
bash ${CLAUDE_SKILL_DIR}/../../scripts/api/record-version.sh {version}
```

### ステップ 3: 完了メッセージ

「v{version} を確認済みにしました。次回から通知されません。」と表示する。
